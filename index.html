点分类的时候才发现，去年10月份最后一次更新写php的事，还有2014年底还在尝试Python，因为找的课程太难，我还放弃了，大概只学了一点来着，我那时候完全没准备再尝试，都从我的梦想里划掉了233。哈哈，然后找了份要死的工作233，讨厌的要死吧，我是不喜欢工作的内容来着，大概就制作xxx的时候觉得有意思，还有教人家做xxx时也不错，其他不好玩，其他有时候占一整天的6/7吧。不管这些，这些只是推进我想改变职业的一个契机，7月份已经开始学php了。php大家说简单来着，那时候不太懂，也不知道什么比较热门，反正如果知道大概会学ios，但好像今年听说需求不是那么大了，我也更想学swift来着，oc听说挺难，java我是没那么大兴趣，但好像也能做很多事。等我学完swift，大概会有下一步打算。

现在会的：

  • html,css
  • php,mysql
  • js,jquery
  • ajax
  • http协议
  • xml
  • git
  • thinkphp
  • phpcms
  • yii
    里面大部分都在1，2月份就都看视频看完了，但我觉得我有好多没理解深入吧，像js，最近做项目时才明白了很多关键点。
    现在六月了，工作换了两个，现在是第三个，大概算是稳定下来了。但我在找第二个的时候就比较沉着了。大概感觉到，只要肯好好和我聊的基本都喜欢我[手动滑稽]，哈哈，虽然有点夸张，但我学php以来大概算是我受到表扬最多的，我整个人因为这一年发生的好多事，自己心理上都改变了很多。真的很高兴学了编程，现在我会运动来保持，改善自己的身体，为了更长久的做这行，当然做什么工作，都要注意身体。而且因为工作性质的缘故还有我自身的缘故，我大概只要有工作就不用太担心钱的事。我只对电子类的产品没抵抗力，吃的穿的貌似我都不是很费。
    接下去想学linux等，变得更厉害，想自己建站来着，自己做个网站玩玩，想想就好开心，还有我还想等学了ios就做app游戏，没法做太大的，大概是做那种本身就耐玩，规则好玩的游戏吧
    嘛，这次就到这~
