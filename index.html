gitbook

写书利器。

推荐软件

  1. sublime 编辑非md文件
  2. typora 编辑markdown文件
  3. node.js 安装gitbook的软件 安装后关掉cmd重开

安装

  • ps win用户cmd使用管理员身份运行

  1. 安装nodejs https://nodejs.org/zh-cn/

  2. npm install gitbook-cli -g

  3. 在需要建gitbook的文件夹下 gitbook init
  4. gitbook serve 访问 http://localhost:4000
  5. gitbook build编译，把_book放到服务器,就能访问

个税

纳税人识别号： 身份证号

税构成

工资-固定减5000-专项扣除（五险一金）-5项附加扣除 （新的！）=应交税金额

以年来计算税的情况，若整年的5000*12额度没有超，某月多交，可次年去退税。

6项扣除

子女

3岁到博士 ，可减。

夫妻可一方加，或一半一半。

继续教育

职业资格证书，需要国家级的。

住房贷款

首套房，合同或凭证。

房租

本地有房不能享受。

父母赡养

每一个满60岁，减1000。

大病医疗

医保目录里，自付一年超15000，8万内。留好黄的付账单。

  1. 公司（保留5年信息）
