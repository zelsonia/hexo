gitbook

推荐软件

  1. sublime 编辑非md文件
  2. typora 编辑markdown文件
  3. node.js 安装gitbook的软件 安装后关掉cmd重开

安装

  • ps win用户cmd使用管理员身份运行

  1. 安装nodejs https://nodejs.org/zh-cn/

  2. npm install gitbook-cli -g

  3. 在需要建gitbook的文件夹下 gitbook init
  4. gitbook serve 访问 http://localhost:4000
  5. gitbook build编译，把_book放到服务器,就能访问

error

gitignore加了的话,如果加了有md的文件。 build时md无法生成。

服务器部署(docker)

  1. docker-compose.yml里加上新端口
  2. nginx config添加_book所在位置的新配置
  3. 上一个nginx配置里 auth basic (http认证) ，访问限制用户认证
  1. passwd文件 slicejobs:xxxxxx 加密方式：nginx 的 http auth basic 的密码是用 crypt(3) 加密的，而apache是md5加密。
